SOFTWARE

 

Sistema Operativo

El sistema operativo requerido para cada computadora es Windows 10, esto debido a que Windows 7 ya no cuenta con soporte por parte de Microsoft y Windows 8 u 8.1 son muy difíciles de usar debido a su interfaz gráfica.

El costo de cada licencia de Windows 10 en su versión Home es de $4,399.00 y en si versión Pro es de $6,399.00, está información fue tomada de la página oficial de Microsoft.

 

Paquetería de oficina

Cada computadora debe contar mínimo con el Software necesario para tareas de oficina, es por ellos que utilizaremos la paquetería de Microsoft Office.

Tenemos 2 alternativas, un plan familiar que nos permite hacer uso de este Software en un máximo de 2 a 6 equipos, su costo es de $1,749.00 al año, además tenemos el plan personal que solo permite un solo equipo, su costo es de $1,299.00 al año.

Dada la información deducimos que nos conviene mas hacer uso del plan familiar.

 

Otras alternativas

Debido a los altos costos que se deben pagar por el uso de las licencias podemos hacer uso de varias alternativas, como por ejemplo podemos hacer uso del software libre.

Para el sistema operativo podemos hacer uso de la distribución de Linux Ubuntu, el cual es un Sistema Operativo muy básico, intuitivo y fácil de usar para los principiantes en las distribuciones Linux.

Para el uso del Software podemos hacer uso de la paquetería de Libre Office, el cual cuenta con herramientas similares a las de Microsoft Office.
